Detailed Description

Umbrella header for all qdtsne files.

This header is effectively an alias for tsne.hpp. It is provided just to allow developers to #include "qdtsne/qdtsne.hpp" without thinking too hard about the code-containing headers. Of course, #include qdtsne/tsne.hpp also works but is less obvious.
